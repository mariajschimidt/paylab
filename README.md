**Descrição:**
O **PayLab** é uma plataforma web intuitiva e responsiva projetada para simplificar cálculos financeiros e trabalhistas essenciais. Com uma interface moderna baseada na estética *Glassmorphism* (efeito de vidro), o projeto oferece ferramentas precisas para o cálculo de horas extras, projeções de aumento salarial e estimativas de rendimento anual. O objetivo é eliminar a complexidade de cálculos manuais, proporcionando clareza e precisão para o usuário final.

## Funcionalidades
- **Cálculo de Horas Extras:** Processamento automático de horas excedentes com suporte a adicionais de 50% (dias úteis) e 100% (finais de semana e feriados).
- **Simulador de Aumento Salarial:** Cálculo do novo salário bruto a partir do percentual de reajuste informado.
- **Projeção de Salário Anual:** Estimativa do montante acumulado de ganhos ao longo de meses trabalhados.
- **Interface Responsiva:** Layout totalmente adaptável para dispositivos móveis, tablets e desktops.
- **Design Glassmorphism:** Experiência visual premium com uso de desfoque de fundo, transparências e sombras suaves.
- **Formatação de Moeda:** Exibição dos resultados automaticamente formatados para o padrão brasileiro (R$ - BRL).
